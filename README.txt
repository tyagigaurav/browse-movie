Directions for use:

Code Overview:
Code is present in the browse-movie folder,for referring code structure please refer src folder.
Please refer dist folder for production code(npm run prod can be used to create dist folder)

Requirements: Please install node.js and npm to start the server

Steps:
1)Write npm install from \browse-movie
2)After installing node modules by using above command start the server with npm run dev command
3)Node server will be up on http://localhost:3000/#!/app/home url.
4)There are 2 urls for application /home and //movie-filter.
